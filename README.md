- Used technologies:
  - React
  - TypeScript
  - React Hooks
  - Axios
  - CSS
    
# Dependencies
- Node v16.14.2 and higher
- NPM v8.5.0 and higher

# Installing
* Fork and clone this repository
* Run `yarn` in your terminal
* Run `yarn start`
